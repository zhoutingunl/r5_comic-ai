# 小说转漫画生成器 📚➡️🎨

一个**纯前端**应用，使用 **Vercel AI SDK** 和阿里云 DashScope AI 服务，能够将小说文本智能转换为精美的漫画面板。
相关文件内容都在 `docs/` 目录下。

## ✨ 特性

- 🎯 **纯前端架构** - 无需后端，直接在浏览器中运行
- 🔑 **用户自主配置** - 使用自己的 API Key，数据完全私密
- 🚀 **快速部署** - 可部署为静态站点，零服务器成本
- 🎨 **多种漫画风格** - 支持日漫、美漫、韩漫等多种风格
- 👥 **角色管理** - 创建和管理角色库，支持三视图生成
- ⚡ **并发控制** - 智能任务队列，批量生成漫画面板

## 🛠️ 技术栈

- **框架**: React 19 + Vite 6
- **AI SDK**: Vercel AI SDK (`ai` package)
- **AI 服务**: 阿里云 DashScope (Qwen-Plus, Qwen-Image, Qwen-Image-Edit)
- **语言**: TypeScript
- **样式**: Tailwind CSS + ShadcnUI
- **验证**: Zod
- **本地存储**: IndexedDB + localStorage
